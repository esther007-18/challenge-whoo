# challenge-whoo


This game is meant for all ages to mentally challenge players into expanding their critical thinking abilities. The game is structured with the player starting the game by clicking the start button and the question, player icons and the textbox all show up in that one frame. The player enters the their answer in the textbox and the progame decides whether or not the answer is correct which could affect the ranking of the player ultimately leading to their demise by the villian character.
